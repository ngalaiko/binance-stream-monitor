# Binance stream monitor

## Configuration

1. Command line arguments

| Command line            | Default                  | Description                    |
| ----------------------- | ------------------------ | ------------------------------ |
| verbose                 | false                    | Enable verbose logging         |
| alert                   | []                       | Enable verbose logging         |

## Running

1. Locally
```
$ cd backend && \
    go run ./cmd/monitor/main.go
```
